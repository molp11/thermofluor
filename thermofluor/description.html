An Interactive Fitting Tool for Thermofluor Experiments

Use the green buttons to import raw data from your computer and export fitting session from the data table.

Use the selection tools below the green buttons to choose which model fits to display on the heatmap, data table and sample plot.

Use the figure tools at the upper right of each plot to zoom, pan, save plot to image file, select raw plot data for fitting and hover over the circles on the model fit plot to see sample information.

Select cells in the heatmap and data table to select samples to plot on the right.

Box select x axis ranges on the fit plot to refit the model on a subset of the raw data.

(Note: input data must be in csv format and is expected to be in columns in order (to correctly format the heatmap): [Temperature, A1, B1, C1, D1,..., A2, B2, C2, D2,...] . Some special characters (i.e. greek symbols) may not be supported and could cause an error during file upload. All columns should contain only integer or float values.
